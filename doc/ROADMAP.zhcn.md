# 路线图

路线图由若干里程碑构成，每个里程碑为一个自然月（其间可能有假期，历时3至4周），以元素周期表顺次命名：

- 大多数里程碑结束时会发布一个新版本，版本号遵循[semver规范](https://semver.org/)
- 开发默认遵循`Explore-Execute-Evolve`节奏，即探索、执行、演进
